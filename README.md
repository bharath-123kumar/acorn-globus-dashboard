This zip contains a minimal runnable scaffold of the assignment:
- backend (Node/Express + Mongoose)
- frontend (React skeleton)

Instructions:
1. Unzip and open `backend` and `frontend`.
2. In `backend`, set `MONGO_URI` in `.env` or use local MongoDB.
3. Run `npm install` in both `backend` and `frontend`.
4. Seed DB: `node seed/seed.js` (inside backend).
5. Start backend: `npm run dev` (requires nodemon) or `node server.js`.
6. Start frontend: `npm start`.
